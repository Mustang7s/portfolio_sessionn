## Design Context

### Users
- **Target Audience**: High-end agencies, tech recruiters, tech-forward potential employers.
- **Job to be Done**: Showcase technical mastery, innovative thinking, and professional reliability through a distinctive, performant portfolio.

### Brand Personality
- **Voice/Tone**: Innovative, Sharp, Forward-thinking.
- **3-Word Personality**: "Sophisticated, Precise, Kinetic".
- **Emotional Goals**: Confidence, delight through micro-interactions, technical wonder.

### Aesthetic Direction
- **Visual Tone**: Minimal yet bold, technical sophistication, organic fluidity (Vercel/Linear/Stripe inspired).
- **Theme**: Dark mode as default (preferred for developer portfolios), with a smooth light mode toggle.
- **Palette**: #222222 (Graphite), #FAF3E1 (Ivory), #F5E7C6 (Beige), #FF6D1F (Accent Orange).
- **Typography**: Instrument Serif for dramatic display, Google Sans for clean UI, Montserrat for modern accents.
- **Anti-references**: Cluttered landing pages, generic Bootstrap templates, overly playful/cartoonish aesthetics, excessive/slow animations.

### Design Principles
1. **Precision in the Micro**: Every interaction should feel intentional and fast (Linear-style).
2. **Whitespace as a Feature**: Use generous spacing to create a high-end, uncluttered feel (Stripe-style).
3. **Organic Fluidity**: Animations should feel like physical objects with mass and momentum, not just linear transitions.
4. **Typography-First**: Use bold serif headers to create strong visual hierarchy and character.
5. **Technical Sophistication**: Subtle touches like noise overlays and cinematic transitions to signal technical depth.
